# Adhan Audio Assets

This directory should contain the following audio files for prayer notifications:

## Required Files

1. **adhan-full.mp3** - Full traditional Adhan (2-3 minutes)
2. **adhan-short.mp3** - Short Adhan excerpt (30-60 seconds)
3. **beep.mp3** - Simple notification beep (< 5 seconds)

## File Format Requirements

- Format: MP3
- Sample Rate: 44.1kHz or 48kHz
- Bitrate: 128kbps minimum
- Mono or Stereo

## Licensing

Ensure all audio files are properly licensed for use in this application.
Free sources for Adhan audio:
- Internet Archive (archive.org)
- Free Islamic Audio Libraries
- Creative Commons licensed recordings

## Platform Notes

### Android
- Custom sounds work via notification channels
- Files should be < 5MB for best performance

### iOS
- Custom notification sounds have limitations
- System will use default sound, but in-app playback works
- Files should be < 5MB
