You are working on **Adhan App**, a privacy-first Islamic companion application.

This app includes:
- Prayer Times & Adhan
- Quran (Arabic + translations)
- Learn (revert-friendly learning)
- Qibla
- Hadith
- AI Smart Search (Quran & Hadith)
- Hijri Calendar & Ramadan countdown
- Optional Authentication + Premium features

========================
NON-NEGOTIABLE RULES
========================
• Prayer times are calculated OFFLINE using astronomical formulas.
• Quran Arabic text MUST be:
  - UTF-8
  - unmodified
  - from a trusted source
  - clearly attributed in Credits & Sources.
• AI is EDUCATIONAL ONLY:
  - No fatwa
  - No halal/haram rulings
  - No opinions
• AI must:
  - cite Quran ayah or Hadith references
  - or explicitly say “I am not sure”.
• Icon-first UX:
  - No “Close” text buttons
  - Icons only (back, close, search, mic, send)
  - Accessibility labels must be translated.
• All screens MUST use SafeArea via <Screen> component.
• Theme-aware everywhere:
  - Light / Dark / System
  - Font scaling supported.
• All user-facing strings MUST use i18n:
  - English, French, Swahili, Arabic.
• Minimal diffs:
  - Do NOT rewrite architecture unless explicitly requested.
• No heavy dependencies unless absolutely necessary.
• Always preserve existing APIs and data flow unless told otherwise.

========================
UX PRINCIPLES
========================
• Calm, respectful, non-gamified design.
• Revert-friendly and beginner-friendly.
• Clear states (location OK/error, notifications on/off).
• Offline-first wherever possible.
• No visual clutter.
